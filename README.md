## サイト概要
### サイトテーマ
関心のあるアロマ・ハーブの講座を検索したり、その講座への問い合わせや申し込みができたりするコミュニティサイト
​
### テーマを選んだ理由
アロマや手作りすることが好きで、ワークショップや講習会に参加することも、母と一緒に自ら企画・開催することもあるのですが、「これだ」と思うものを見つけたり、逆に周知したりすることが大変だと感じていました。  
現職で講習会の企画から開催までを担当する中で、お客様から「もう少し見つけやすくて、そこから直接申し込めたら便利なのにね」というお声をいただくこともあり、検索も問い合わせや申し込みもできるサイトがあれば便利だと考えました。  
検索や問い合わせ・申し込みのしやすいサイトであれば、アロマ・ハーブや手作りに関心がある人だけでなく、何かを始めてみたい、やってみたいという人のイベント参加への一歩の役に立つと思い、また、母の夢である「自分の教室の講座に参加してもらう」ことにも、同時に役立てると思い、このテーマを選びました。
​
### ターゲットユーザ
- アロマ・ハーブに関心がある人
- 心身の健康に関心がある人
- 手作りが好きな人
- 何かを始めたいと考えている人
​
### 主な利用シーン
- アロマ・ハーブの講座を検索するとき
- 手作り体験をしたいとき
- アロマ、ハーブ関連の資格を取りたいとき
- アロマ、ハーブの活用方法を知りたいとき
- 講座に関する質問があるとき
- 講座への申し込みをしたいとき
​
